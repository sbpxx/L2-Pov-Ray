# L2-Pov-Ray

> Chess+ animation made with POV-Ray — 2nd year university project.

## Description

Image synthesis project created in L2 (2nd year of university). The goal was to build an animated 3D scene of an enhanced chessboard (*Chess+*) using **POV-Ray SDL** (Persistence of Vision Raytracer), a scene description language for rendering photorealistic images.

## Repository structure

```
PROJET_POVRAY/             # POV-Ray source files (.pov, textures, etc.)
RAPPORT_POVRAY_SIDALI.pdf  # Project report
```

## Tech stack

| Tool | Usage |
|---|---|
| POV-Ray SDL | Scene description language |
| POV-Ray | Ray tracing render engine |

## Requirements

- [POV-Ray](https://www.povray.org/) installed on your machine.

## Render

```bash
povray PROJET_POVRAY/scene.pov
```

> Replace `scene.pov` with the actual main file name.
